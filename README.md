# Светофор Мебели

Простой одностраничный сайт для мебельного магазина.

## О проекте

Это статический landing page с информацией о мебели, категориях товаров и преимуществах магазина. Проект оформлен на HTML и CSS, подключает шрифты Google и иконки RemixIcon.

## Структура проекта

- `index.html` — основная страница сайта.
- `assets/style.css` — стили для страницы.
- `fonts/remixicon.css` — CSS для иконок RemixIcon.

## Использование

1. Откройте `index.html` в браузере.
2. Или запустите через локальный сервер, например с помощью VS Code Live Server.

## SEO

В `index.html` добавлены базовые SEO-мета-теги:

- `description`
- `keywords`
- `author`
- `robots`
- `theme-color`
- Open Graph (`og:*`)
- Twitter Card (`twitter:*`)

## Рекомендации

- Замените `og:url` на фактический URL сайта перед публикацией.
- При необходимости добавьте favicon и полные тексты для разделов.
- Если планируется поддержка нескольких страниц, перенесите общие мета-теги в шаблон.
